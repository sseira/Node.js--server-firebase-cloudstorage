# simple-node-server

A barebones Node.js app using [Express 4](http://expressjs.com/).


## Connections:


### firebase

### http requests 

### Google Maps API

### Arduino/RasberryPI

### Slack

### Framer.js

## Running Locally

```sh
$ npm install
$ nodemon ./server.js localhost 5000
$ npm start
```
